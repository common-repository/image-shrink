You will get the resize of upload image, whichever is larger of the width or height

== Description ==
This plug-in, if you upload a large image file than the size you have set, you can save by reducing the image file to that size.
It will be published in the necessary and sufficient image size to blog image data have been recorded with smartphone or camera having a resolution higher, without having to file repairing.

== Installation ==
1. Upload "imageshrink" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Set Maximum image size at initial on setting.
4. Upload image datas (jpeg,gif,png) are changed imagesize.
